# Dotfiles


## Usage

Install

``` sh
git clone https://github.com/anufriev-dev/dotfiles.git --recursive
cd dotfiles && ./install

```

Any new changes can be retrieved from the repository and installed using the following commands:

``` sh
git pull
./install
```
